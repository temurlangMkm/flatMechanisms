MechIsCool (KinematicVis)

Данный проект представляет собой десктопное приложение для визуализации и симуляции кинематики плоских механизмов. 
Основная цель программного продукта — предоставление инструмента для быстрой проверки кинематических схем посредством 
конфигурационных файлов, исключая необходимость модификации исходного кода.

Архитектура приложения базируется на платформе Java 21 с интеграцией графического фреймворка JavaFX версии 21.0.6 
для отрисовки пользовательского интерфейса. Обработка конфигурационных файлов формата JSON реализована средствами библиотеки Jackson 2.18.3.
Процесс сборки и управление проектными зависимостями автоматизированы через Apache Maven.

Настройка механизмов осуществляется строго через внешние JSON-файлы. Модель данных требует указания базовой угловой скорости кривошипов, 
определения массива узловых точек и массива соединительных звеньев. Система поддерживает неподвижные опоры, свободные шарниры, ползуны с заданными направляющими осями
и жестко закрепленные точки на теле существующих звеньев. Кинематические связи формируются с помощью ведущих кривошипов и ведомых шатунов. Исчерпывающая спецификация параметров и правила создания конфигураций задокументированы в файле JSON_MMANUAL.md.

Для развертывания проекта в локальной среде требуется установленный набор инструментов Java Development Kit 21. 
Запуск приложения в режиме разработки выполняется консольной командой mvn javafx:run. Проект конфигурации Maven включает плагин для создания автономных исполняемых пакетов.
Сборка дистрибутива под текущую операционную систему инициализируется командой mvn jpackage:jpackage. 
Сгенерированные установочные файлы помещаются в директорию target/dist. Созданный дистрибутив включает в себя интегрированную среду выполнения Java, 
обеспечивая независимость от системного окружения конечного пользователя.

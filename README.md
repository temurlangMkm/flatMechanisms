# MechIsCool

MechIsCool - desktop-приложение на Java 21 и JavaFX 21 для визуализации и симуляции плоских кинематических механизмов.

Механизм задается внешним JSON-файлом. Конфигурация описывает:

- неподвижные опоры;
- свободные шарниры;
- ползуны с направляющими;
- точки, закрепленные на звене;
- ведущие кривошипы и ведомые жесткие звенья.

## Документация JSON

- `src/main/resources/JSON_MANUAL.md` - строгий справочник формата JSON для пользователя.
- `src/main/resources/JSON_AI_AGENT_GUIDE.md` - подробная инструкция для ИИ-агентов, которые генерируют JSON-конфигурации.

## Запуск в разработке

Требуется JDK 21.

```bash
mvn javafx:run
```

## Проверка сборки

```bash
mvn -q -DskipTests compile
```

## Сборка дистрибутива

```bash
mvn jpackage:jpackage
```

Результат создается в `target/dist`.
